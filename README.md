## Tech Stack

Following latest Android Norms

- Kotlin DSl used (with version catalog .toml file)
- Ime Service in Jetpack Compose
- Accessibility service
- Ktor-Client used.
- Account Manager for Android.
- Node JS for Backend
- Llama Index
- MBTI Personality Prediction Model
